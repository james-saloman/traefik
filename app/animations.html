Traefik isn't just for simple web apps — it's the front door for Inductive Automation's Ignition platform too. Here's what actually happens, hop by hop, when someone opens a Perspective session and the Ignition Gateway is running inside a Docker container behind Traefik.

What's happening at each hop

The response — the rendered Perspective session or Vision applet — travels back through the exact same path: Ignition Gateway → Docker network → Traefik → browser. The browser never talks to the container directly; Traefik and Docker's internal networking are the only things that know the container's real IP and port.
